# Лига триатлона · IX этап · Черкесск — 360° галерея

Лендинг со сферической фотогалереей. Photo Sphere Viewer 5.7 + Three.js, подключаются с CDN.
Сборки нет — это статические файлы.

## Структура

```
index.html
panoramas/
  pano-01.jpg     6000×3000  панорама (equirectangular)
  pano-02.jpg
  thumb-01.jpg     640px     превью для плитки
  thumb-02.jpg
```

## Локальный запуск

Открыть `index.html` двойным кликом **не получится** — ES-модули требуют HTTP.

```bash
cd <папка проекта>
python3 -m http.server 8000
# затем http://localhost:8000
```

## Как добавить панораму

1. Положите JPEG в `panoramas/`. Требования:
   - **строго 2:1** (equirectangular), например 6000×3000
   - до ~2 МБ, иначе долго грузится на мобильных

   Ресайз исходника 8000×4000 с Insta360:
   ```bash
   convert original.jpg -resize 6000x3000 -quality 82 -strip panoramas/pano-03.jpg
   ```

2. Сделайте превью — кроп интересного участка, не всю сферу (иначе плитка выглядит искажённой):
   ```bash
   convert original.jpg -crop 2400x2400+2800+900 +repage -resize 640x \
     -quality 80 -strip panoramas/thumb-03.jpg
   ```
   `+2800+900` — смещение кропа. Подберите под кадр.

3. Добавьте объект в массив `PANORAMAS` в `index.html` (ищите комментарий «ДАННЫЕ ПАНОРАМ»):

```js
{
  id: 'p3',
  src:   'panoramas/pano-03.jpg',
  thumb: 'panoramas/thumb-03.jpg',
  title: 'Велоэтап',
  sub:   'Разворотная точка на трассе',
  tag:   'track',          // start | team | track
  tagLabel: 'Трасса'
}
```

Счётчик в шапке и нумерация плиток обновляются сами.

## Заглушки

`const PLACEHOLDER_SLOTS = 2;` — количество пунктирных «мест под кадр» в конце сетки.
Поставьте `0`, когда галерея заполнена.

## Фильтры

Категории заданы в HTML в блоке `.filters` через `data-filter`. Чтобы добавить категорию —
допишите кнопку и используйте то же значение в поле `tag` у панорам.

## Публикация

**GitHub Pages** — как lapin-lab.ru: залить репозиторий, включить Pages, ветка `main`, папка `/`.

Панорамы весят ~2 МБ каждая. Если их станет больше 15–20, лимит репозитория Pages (1 ГБ)
не проблема, но трафик — да. Тогда лучше вынести `panoramas/` на объектное хранилище
(Yandex Object Storage) и указать полные URL в `src`.
